생활코딩 HTML 사전링크

태그는 문법의 시작을 알리고 속성은 문법의 시작 안에 부수적으로 넣는 용도입니다.
대부분의 태그는 속성없이 사용가능하지만 속성이 있어야만 사용가능한 태그도 있습니다.

HTML의 사용법을 정의 해놓은 페이지 입니다.

strong 은 사이에 있는 문자를 강하게 표현해주는 or 굵게 표현해주는 태그 입니다.

h1 태그는 제목이라는 의미 head의 의미를 담고있으며
숫자가 1,2,3,4,5,6 이 있으며 숫자가 작을수록 크고 굵게 표현되며 줄바꿈을 알아서 하는 태그 입니다.

a 태그는 anchor라는 이름을 딴 태그이며 링크를 걸어주는 태그 입니다. 하지만 이것만으로는 정보가 부족합니다.
a태그는 href = "대상주소" 즉 하이퍼 링크라고 하는 이름의 속성으로 주소를 걸어줘야 작동합니다.
또한 target = "_blank" 속성을 이용해 새로운 페이지로 주소를 열수도 있습니다.

속성 title="주석" 태그 안에 집어넣어 마우스를 올림으로써 주석을 볼 수 있습니다.

li 태그는 각각의 항목을 하나하나씩 나열해주는 태그입니다.
